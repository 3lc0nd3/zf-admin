ZF-Admin

ZF-Admin uses Gulp to build its assets.

Available Gulp tasks are:

  • "gulp help". This lists all available tasks.
  • "gulp build-js". Builds all JS assets.
  • "gulp build-sass". Builds all SASS assets.
  • "gulp prepare-example-assets". Copies main ZF-Admin assets(JS and CSS) to the example folder.
  • "gulp prepare-default-images"
  • "gulp copy-fonts"
  • "gulp lint-js"

defined in an Array named jsFiles. You can selectively comment out the assets you don't need. Even all the Zurb Foundation plugins are listed.

These are just recommendations. You can use any build system you feel is better for your project.
